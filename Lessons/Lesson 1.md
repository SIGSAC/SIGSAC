## Github Basics

### The golden commands:

1. `git pull --rebase`
2. `git status`
3. `git add [file]`
4. `git commit -m "[comment]"`
5. `git push`

We must first add our email and name in order to commit and push files.

```
git config --global user.email "paul.tan@westpoint.edu"
git config --global user.name "PillowFood"
```
